# Daily Brief

Your morning snapshot of what changed in your Mixpanel project.

## What it covers

1. **Anomalies**: Metrics that moved significantly (>10% change)
2. **Experiments**: Active experiments and recent completions
3. **Data Quality**: New issues or warnings
4. **Key Metrics**: DAU, WAU, MAU, key conversions

## Output Format

Present as:

**🚨 Attention Needed**
- Critical anomalies or failed experiments

**📈 Metrics Update**
- DAU, WAU trends
- Key conversion rates

**🧪 Active Experiments**
- Running experiments and progress

**✅ All Clear**
- No critical issues detected

## Best Practices

- Keep concise - this is a quick morning scan
- Link to relevant reports
- Flag only actionable items
- Include % changes for context
