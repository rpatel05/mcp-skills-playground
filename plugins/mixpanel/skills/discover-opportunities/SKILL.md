# discover-opportunities

Discover product opportunities by synthesizing multiple data sources.

## What it analyzes

1. **Analytics**: Underperforming funnels, drop-offs, declining trends
2. **Experiments**: Failed tests that revealed insights
3. **Session Replays**: Common friction patterns
4. **Feedback**: Recurring user requests or complaints

## Output Format

**Product Opportunities**

**High Priority 🔴**

**1. [Opportunity Name]**
- **Impact**: [High/Medium/Low]
- **Effort**: [Est. effort]
- **Evidence**:
  - Analytics: [Metric/finding]
  - Replay: [Pattern observed]
  - Feedback: [User quote]
- **Recommendation**: [Specific action]

**Medium Priority 🟡**

[Similar format]

**Low Priority 🟢**

[Similar format]

## Best Practices

- Prioritize by impact × confidence
- Include quantitative evidence
- Make recommendations specific
- Link to supporting data
- Focus on actionable opportunities
